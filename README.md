# E-FamiTracker Builds
Windows builds of Eulous's E-FamiTracker


The original repo is gone but GitHub user Milkve still has it but in source form only: https://github.com/Milkve/E-FamiTracker.
I downloaded that source and built it. (Built with Windows SDK 10.0, so sadly Windows XP will not work.)

### Backstory on E-FamiTracker
E-FamiTracker (Extended FamiTracker) was a project from Eulous to make the ultimate FamiTracker with: 
- 6581 SID chip (C64)
- Philips SAA1099 chip (SAM Coupè, no instrument editor)
- *AY-3-8910 chip (ZX Spectrum 128k, see the Sunsoft 5B)
- AY8930 chip (Covox Sound Master)
- *SN76489  chip (Sega Master System, see SnevenTracker)
- *T6W28 chip (NeoGeo Pocket Color, extended SN76489)
- *TIA chip (Atari 2600, 7800)
- *POKEY chip (Atari 5200)
- 5E01 chip (Fantasy chip, works via expansion port, adds 2 noise and 2 DPCM channels)
- a LOT of extra features (check https://github.com/Milkve/E-FamiTracker for them all)

*(Chips with * are not implemented)*

All of the unimplemented chips are also in Furnace so if you want to use those, use Furnace instead.

But eventually she stopped the project. She also took the GitHub link offline but GitHub user Milkve archived it (source only).
I have built that source code and I'm uploading it here.
